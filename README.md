# 🤖 Smart AI Chatbot (Streamlit + Groq)

A modern AI-powered chatbot built using **Streamlit** and **Groq API**.
This chatbot supports real-time conversation, voice input, and fast AI responses.

---

## 🚀 Features

* 💬 ChatGPT-like user interface
* ⚡ Ultra-fast responses using Groq API
* 🎤 Voice input (Speech Recognition)
* ⌨️ Typing animation effect
* 🧠 Context-based conversation
* 🆓 Free API (Groq)

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Groq API (LLaMA 3.1 model)
* SpeechRecognition
* PyAudio

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/chatbot-project.git
cd chatbot-project
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

---

## 🔑 Setup API Key

Create an account at: https://console.groq.com

Then set your API key:

```bash
setx GROQ_API_KEY "your_api_key_here"
```

Restart terminal after this.

---

## ▶️ Run the App

```bash
streamlit run chatbot.py
```
